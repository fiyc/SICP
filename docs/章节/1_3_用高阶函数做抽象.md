## 1.3 用高阶函数做抽象  
> 以过程作为参数或是返回值得过程, 被称为高阶过程. 本节将展示高阶过程如何能称为强有力的抽象机制, 极大的增强语言的表述能力.  

### 1.3.1 过程作为参数  
下面使用3个不同的求和过程, 将他们转为一个高阶过程  

1. 计算从a到b的各整数之和:  
```lisp
(define (sum-integers a b)
	(if (> a b)
		0
		(+ a (sum-integers (+ a 1) b))))
```  

2. 计算给定范围内的整数的立方之和:  
```lisp
(define (sum-cubes a b)
	(if (> a b)
		0
		(+ (cube a) (sum-cubes (+ a 1) b))))
```

3. 计算下面序列之和:  
1/(1*3) + 1/(5*7) + 1/(9*11) + ...  
```lisp
(define (pi-sum a b)
	(if (> a b)
		0
		(+ (/ 1.0 (* a (+ a 2))) (pi-sum (+ a 1) b))))
```
